Features
--------
- acquire and set environment variables required by mflux-server
- cache the environment variables


Quickstart
----------
Install mflux-ai

    pip install mflux-ai

Import mflux_ai

    Import  mflux-ai

    mflux_ai.set_env_vars(token="9s9kQ0D86wWKUHdPMj0HHA")
