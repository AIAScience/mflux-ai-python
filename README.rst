Open source code for the mflux-ai python library.

Features
--------
- Acquire and set environment variables required by mflux-server

Quickstart
----------
Installation

    pip install mflux-ai

Usage

    import mflux_ai

    mflux_ai.set_env_vars(token="insert_your_token_here")
